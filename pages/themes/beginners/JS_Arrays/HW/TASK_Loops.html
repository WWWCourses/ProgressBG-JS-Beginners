Забележки:

Моля, спазвайте следните изисквания при реализацията:

  • Създайте папка xyz_HW_Loops, като заместите 'xyz' с вашите инициали. Например, аз бих работила в папка iep_HW_Loops.
  • В така създадената папка създайте един html файл с име: HW_Loops.html. За всяка задача използвайте отделен .js файл (в същата папка) с име съвпадащо с името на дадената задача, който ще включвате към HW_Loops.html.
  • Архивирайте папката xyz_HW_Loops и изпрате по мейл съответния архивиран файл.
